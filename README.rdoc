ABOUT:

  The Chefbuddy app is a recipe app that allows users to sign up, create  recipes,save and share recipe 
  by email.

TECHNOLOGIES USED:
  
      *FRONT-END: CSS3,Bootstrap-Saas
      *BACK-END: Ruby, Ruby on Rails, API integration using Mandrill-Api, Ruby gems used include-Cocoon, Devise,
                 Simple-form and paperclip
      
FEATURES:

    *Create User account
    *Create recipes and save
    *Upload recipe images
    *Share recipe with friends by email
